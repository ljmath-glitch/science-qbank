詳解匯入

把含有詳解的 CSV／Excel 貼上或上傳，系統會依「考試名稱」＋「題號」自動比對到題庫中已存在的題目，只更新詳解欄位，其他資料不變（找不到對應題目的列會略過並列在結果訊息裡）。
檔案第一列（標題列）必須包含這三個欄位名稱：考試名稱　題號　詳解（順序不拘，可以有其他欄位，會自動忽略）。
💡 不知道怎麼準備這份檔案？兩種方式：
① 用 AI 生成：先在題目列表勾選要做詳解的題目，點工具列「📋 詳解製作指令」，複製指令給主管的 AI，AI 照著格式回傳的 CSV 就能直接貼來這裡匯入。
② 給人手動填寫：點工具列「📤 詳解匯出Excel」，會下載一個 ZIP，裡面有已經填好考試名稱／題號／題目／答案的 Excel，含圖的題目圖片也會一起打包在 images 資料夾裡（不用另外找紙本考卷），交給主管填好「詳解」欄後，把裡面的 Excel 上傳到這裡即可（欄位已經對齊，不用調整格式）。
Ollama（本機 AI）使用說明

這是什麼？
用公司或自己 Mac 上跑的本機 AI 模型來生成詳解，不用另外申請 API Key，但需要先透過 Tailscale 連上同一個私人網路。

第一次使用前的準備
1. 到 tailscale.com/download 下載安裝 Tailscale
2. 用公司統一帳號登入（跟公司電腦、其他同事同一組帳號）
3. 登入後這台裝置就會自動加入同一個私人網路，能連到下面的選項

下拉選單三個選項怎麼選：
・大南路櫃2：透過 Tailscale 連公司那台電腦上跑的 AI，前提是那台電腦要開機、Ollama 程式要在執行中，其他都不用管
・福大 Mac：自動依序嘗試「本機 → 同 Wi-Fi → Tailscale」三種方式連福大 Mac 的 Ollama，不管是自己在那台 Mac 上用、跟它在同一個 Wi-Fi、還是透過 Tailscale 遠端連，都能自動找到能用的那一種，不用手動切換
・其他：手動輸入網址。可用逗號分隔多個網址（例如 http://100.72.250.96:11434,http://100.127.211.80:11434），系統會依序嘗試，第一個連不上會自動切下一個

常見問題
按「AI 生成詳解」出現連線錯誤 → 通常是 Tailscale 沒登入、電腦沒開機、或 Ollama 程式沒在跑，逐一檢查即可。
